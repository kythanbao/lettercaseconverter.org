_Last updated_: February 22, 2026

This page is for users who want to opt out of sale/share style processing related to targeted advertising.

## 1. Use site privacy controls

Open the **Cookie settings** control and disable:

- Advertising
- Analytics (optional but recommended for strict privacy mode)

These settings apply to this browser/device.

## 2. Global Privacy Control (GPC)

If your browser sends a GPC signal, we treat it as an opt-out signal for optional advertising/analytics processing in our consent layer.

## 3. Submit a formal request

If you want a formal privacy request record (access, deletion, correction, opt-out, appeal), use:

- [/privacy-request](/privacy-request)

## 4. AdSense US state message checklist

In AdSense dashboard, make sure **Privacy & messaging** is configured for US states:

- publish a US state regulations message for this domain
- verify message status is active on production domain
- test rendering with Google Funding Choices test query params when needed

Code support for Funding Choices and consent mode is already enabled on this site, but message creation/publishing must be completed in the AdSense UI.
