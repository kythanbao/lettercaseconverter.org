# Code of Conduct

## Our Standard

We expect all contributors to be respectful, constructive, and professional.

## Expected Behavior

- Be respectful in discussions and reviews.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions.

## Unacceptable Behavior

- Harassment, insults, or discrimination.
- Hostile or intentionally disruptive communication.
- Sharing private data without permission.

## Enforcement

Maintainers may remove comments, close threads, or block contributors for violations.

## Reporting

Report conduct issues to:

- `contact@lettercaseconverter.org`
